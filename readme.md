#H1 *Научиться учиться*
В проекте сайта _"Научиться учиться"_ отображены главные проблемы обучения, техники обучения, принципы обучения, а так же видео для тех, кто любит прокрастинировать.

В данном проекте используются: анимация, флексбокс верстка, встроенные видеофайлы, БЭМ.
Используются языки: HTML, CSS.
Организация файловой структуры: Nested.
